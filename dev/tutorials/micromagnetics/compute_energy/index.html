Calculating micromagnetic energies

Import JuMag

using JuMag

Create a finite difference mesh

mesh =  FDMesh(dx=5e-9, dy=5e-9, dz=5e-9, nx=1, ny=1, nz=1)
FDMesh(5.0e-9, 5.0e-9, 5.0e-9, 1, 1, 1, 1, 1.2500000000000001e-25, [-1; -1; -1; -1; -1; -1;;], false, false, false)

Create a Simulation

sim = Sim(mesh);
[ Info: Standard Sim (CPU) has been used.

Set the saturation magnetization Ms

set_Ms(sim, 8.6e5)
true

Initilize the magnetization to (1,1,1) direction, note that |m|=1 so the magnetization vector m = (1,1,1)/sqrt(3).

init_m0(sim, (1,1,1))
3-element view(::Vector{Float64}, :) with eltype Float64:
 0.5773502691896258
 0.5773502691896258
 0.5773502691896258

Consider the demagnetization energy

demag = add_demag(sim);

Consider the zeeman energy

zeeman = add_zeeman(sim, (0,0,1e5))
JuMag.Zeeman([0.0, 0.0, 100000.0], [0.0], "zeeman")

Calculate the effective field

JuMag.effective_field(sim, sim.spin, 0.0)
0

Calculate the energies

println("Demag Energy: ",sum(demag.energy), " J")
println("Zeeman Energy: ",sum(zeeman.energy), " J")
Demag Energy: 1.93626827216251e-20 J
Zeeman Energy: -7.799337266207139e-21 J
